# muka

> Smart Campus Notification Hub — AI-powered notification filtering for students.

muka sorts your campus notifications into three cognitive zones — **Instant**, **Scheduled**, and **Batch** — so you always know what needs attention right now vs. what can wait.

## Tech Stack

- [Next.js 16](https://nextjs.org) (App Router)
- TypeScript
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Project Structure

```
src/
├── app/          # Routes & API (App Router)
├── components/   # UI Building Blocks
├── hooks/        # Client-side logic
├── lib/          # Server-side utilities
├── types/        # TypeScript interfaces
└── utils/        # Helper functions
```
